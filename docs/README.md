# Documentação do Projeto

Esta pasta reúne os guias auxiliares do projeto para manter a raiz mais organizada.

## Guias Disponíveis

### Copilot e Prompts

- [COPILOT_PROMPTS.md](COPILOT_PROMPTS.md)
  Explica a skill de interface design usada com GitHub Copilot no VS Code, detalha os prompts disponíveis e mostra quando usar cada um.

### Sistema Visual do Workspace

- [.interface-design/system.md](../.interface-design/system.md)
  Registra a identidade visual do catálogo para manter futuras alterações de interface mais consistentes.

### Configuração Inicial

- [SETUP_GUIDE.md](SETUP_GUIDE.md)
  Guia rápido para configurar o projeto, Supabase, variáveis de ambiente e acesso administrativo.

### Funcionalidades Específicas

- [ANALYTICS_SETUP.md](ANALYTICS_SETUP.md)
  Explica como ativar e testar o tracking de interesse via WhatsApp e as métricas do dashboard.

- [PROMOTIONS_SETUP.md](PROMOTIONS_SETUP.md)
  Explica como ativar e usar o sistema de promoções no catálogo e no admin.

## Fluxo Recomendado

1. Comece por [SETUP_GUIDE.md](SETUP_GUIDE.md) se estiver configurando o projeto do zero.
2. Consulte [COPILOT_PROMPTS.md](COPILOT_PROMPTS.md) para usar melhor os prompts do Copilot.
3. Consulte [.interface-design/system.md](../.interface-design/system.md) ao ajustar a interface publica.
4. Use [ANALYTICS_SETUP.md](ANALYTICS_SETUP.md) e [PROMOTIONS_SETUP.md](PROMOTIONS_SETUP.md) quando precisar habilitar esses recursos no Supabase.
